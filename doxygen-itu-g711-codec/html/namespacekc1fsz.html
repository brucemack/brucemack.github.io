int16_t kc1fsz::decode_ulaw ( uint8_t  c)

G.771 uLaw Decoder Function

Parameters
cThe 8-bit character signal.
Returns
A signed 16-bit PCM audio sample. The decoder result is ending up in the high 16 bits of the value so keep dynamic range in mind. Output range is -32K to +32K.
